Term Insurance models analysis

What are the different AI models used?
First, we use the linear regression. For the second model, called “Lasso”, we first transform the data in order to be able to use a polynomial regression on it and then we use the Lasso regression on this data. That’s why we can show the results with different degrees. The third model is the K-Nearest-Neighbors model. The fourth model implied 2 different polynomial regressions, one for the premiums under 1000 and one for the premiums above. The last one, called “polynomial scaled”, is a scaled polynomial regression that we complete with a KNN method when we predict data which don’t belong to the dataset.
How do we plot the learning curves?
There are 3150 data in the dataset. First, we split the dataset in 3: the training set, the validation set and the test set. The training set represents 56% of the dataset, the validation set represents 19% of the dataset and the test set represents 25% of the dataset. The models are only trained on the training set and never with the 2 other sets. The learning curves show the evolution of the precision (R²) with respect to the train set size. On the x-axis, we have the number of data used in the trainset and in the y-axis, we have the precision of the model applied on the validation set and on the test set. Keep in mind that the test and validation set size never change.
